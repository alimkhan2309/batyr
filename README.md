# Batyr

A fast-paced 2D action game built with Phaser 3. Created for the Global Game Jam Astana.

## Live Demo
[PLAY THE GAME HERE](https://lightblue-swallow-505250.hostingersite.com/)

---

## How to Play
* Move: Arrow Keys / WASD
* Jump: Spacebar
* Change Mask: E
* Attack: Right Click
* Objective: Beat Em

## Tech Stack
* Engine: [Phaser 3](https://phaser.io/)
* Bundler: [Vite / Webpack]
* Language: JavaScript
* Art: Aseprite

## Local Installation
If you want to run this project locally:

1. Clone the repo:
   ```bash
   git clone [https://github.com/alimkhan2309/batyr.git](https://github.com/alimkhan2309/batyr.git)

2. Install dependencies: 
  ```npm install```

3. Start the development server:
    `npm run dev`
